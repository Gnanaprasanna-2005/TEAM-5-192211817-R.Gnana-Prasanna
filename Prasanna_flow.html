STEP 1 · EXPRESSION TRANSFORMATION

Strip the time component from CALL_TIMESTAMP to get a pure date for grouping.

OUTPUT PORT · EXPRESSION FORMULA
O_CALL_DATE = TO_DATE( TO_CHAR( CALL_TIMESTAMP, 'DD-MON-YYYY' ), 'DD-MON-YYYY')
-- Removes the time portion from timestamp → pure DATE
STEP 2 · AGGREGATOR TRANSFORMATION
GROUP BY O_CALL_DATE
AGGREGATED OUTPUT PORTS
EXPRESSION
COUNT(DISTINCT CALLER)
→ DAILY_ACTIVE_CUSTOMERS
EXPRESSION
COUNT(CALL_ID)
→ DAILY_CALL_VOLUME
EXPRESSION
MAX(DURATION)
→ PEAK_CALL_DURATION
STEP 3 · EXPRESSION (AFTER AGGREGATOR)

Calculate the average calls per customer using the aggregated values.

DERIVED METRIC · AVG CALLS PER CUSTOMER
AVG_CALLS_PER_CUSTOMER = DAILY_CALL_VOLUME / DAILY_ACTIVE_CUSTOMERS
FINAL PORT → TARGET COLUMN MAPPING
PORT / EXPRESSION → TARGET COLUMN (DAILY_CALL_SUMMARY)
O_CALL_DATE → CALL_DATE
COUNT(DISTINCT CALLER) → DAILY_ACTIVE_CUSTOMERS
COUNT(CALL_ID) → DAILY_CALL_VOLUME
MAX(DURATION) → PEAK_CALL_DURATION
DAILY_CALL_VOLUME / DAILY_ACTIVE_CUSTOMERS → AVG_CALLS_PER_CUSTOMER
